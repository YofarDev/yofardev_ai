================================================================================
PROJECT OVERVIEW
================================================================================

Yofardev AI is a Flutter mobile/web/desktop application featuring an animated AI 
avatar that interacts with users via text/voice. The app uses LLM integration 
(Gemini, OpenAI-compatible APIs) with function calling capabilities, text-to-speech,
and lip-synced avatar animations.

Version: 3.0.0+6
Platforms: Android, iOS, Web, Desktop (Windows, MacOS, Linux)

================================================================================
TECH STACK
================================================================================

Framework: Flutter (SDK >=3.8.0 <4.0.0)
State Management: flutter_bloc (BLoC/Cubit pattern)
Dependency Injection: get_it
Code Generation: freezed, json_serializable, build_runner
Testing: flutter_test, mocktail, bloc_test, golden_toolkit

Key Dependencies:
- LLM: go_router, fpdart (for functional programming)
- Audio: audioplayers, audio_analyzer, flutter_volume_controller
- AI/ML: speech_to_text, supertonic_flutter
- Storage: shared_preferences, path_provider
- UI: lottie, shimmer, google_fonts
- Location: geocoding, geolocator
- Utils: intl, uuid, math_expressions, alarm

================================================================================
ARCHITECTURE SUMMARY
================================================================================

PATTERN: Clean Architecture with Feature-first organization

1. CORE LAYER (lib/core/)
   - Shared services, models, utilities, widgets
   - DI setup (service_locator.dart)
   - Router configuration
   - Theme, colors, constants
   - LLM, Audio, Agent services

2. FEATURES LAYER (lib/features/)
   - Each feature is a self-contained module
   - Follows: data/ в†’ domain/ в†’ presentation/ structure
   - Uses BLoC pattern for state management
   - Multiple cubits per feature for different concerns

3. STATE MANAGEMENT
   - flutter_bloc with Cubit (simpler than full BLoC)
   - Main providers in main.dart via MultiBlocProvider
   - Feature-specific states and cubits

4. DATA FLOW
   - UI в†’ Cubit в†’ Repository в†’ Datasource в†’ External Service
   - Streaming: LLM в†’ StreamProcessor в†’ TTS в†’ Avatar Animation

5. KEY INTEGRATIONS
   - LLM: Supports Gemini, OpenAI-compatible APIs
   - Function Calling: Weather, News, Google Search, Calculator, Alarm, Web Reader
   - Audio: Text-to-speech, audio playback, amplitude analysis
   - Avatar: Lottie animations, custom mouth sync, background/costume switching
